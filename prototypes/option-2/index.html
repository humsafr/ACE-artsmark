Option 2: Structured self-assessment

This prototype explores a structured assessment using multiple-choice questions, indicators and short supporting explanations.

Schools would respond to a small number of questions across each of the eight Artsmark framework criteria, creating a broad and consistent picture of current practice.

What this option is testing

  • Whether multiple-choice questions reduce effort and ambiguity.
  • Whether a structured format creates a sufficiently rigorous assessment.
  • Whether schools can recognise their own practice in the available answers.
  • Whether short explanations can provide the context assessors need.
